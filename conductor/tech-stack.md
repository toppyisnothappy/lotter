# Tech Stack - Lotter

Modern, high-performance stack for retail operations.

## Core Framework
- **Framework**: Next.js 16 (React 19)
- **Language**: TypeScript
- **State Management**: Zustand

## Data & Backend
- **Database**: Local MySQL
- **ORM**: Drizzle ORM
- **Authentication**: Auth.js (NextAuth)
- **Multi-tenancy**: Shared Database with Column Isolation (`organization_id`)

## UI & Styling
- **Styling**: Tailwind CSS v4
- **Premium Features**: Glassmorphism and Micro-animations

## Deployment & Infrastructure
- **Environment**: Local Only / Internal Development

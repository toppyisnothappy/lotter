## Summary

Implementing the POS core engine, local MySQL schemas, and Drizzle migrations as a replacement for the Supabase infrastructure.

## Context

Lotter is a modern retail management platform for inventory, credit sales, and multi-tenant store operations. This track addresses the core backend infrastructure shift to local MySQL to ensure high-performance and data integrity for POS operations and customer debt tracking.

## User Story

As a **Retail Store Owner**, I want a **reliable MySQL backend and Drizzle ORM integration** so that I can **track complex installment debts and high-speed POS transactions with 100% data integrity.**
